Router - Fungsi, Cara Kerja, dan Penerapan

Oleh: Tiaramailani Ruswandi

üìö Materi Komputer Jaringan: Pembahasan lengkap tentang router sebagai perangkat jaringan penting.

1. Pengertian Router

Router adalah perangkat keras jaringan komputer yang berfungsi untuk menghubungkan dua atau lebih jaringan yang berbeda, baik jaringan yang sama media atau berbeda (seperti LAN ke WAN, atau LAN ke internet). Router bekerja pada lapisan Network Layer (Layer 3) dalam model OSI.

2. Fungsi Router

  • Menghubungkan Jaringan Berbeda: Mengirimkan paket data antar jaringan yang terpisah.
  • Mengatur Lalu Lintas Data (Routing): Menentukan jalur terbaik untuk pengiriman data.
  • Menerapkan Keamanan Jaringan: Dilengkapi fitur firewall, NAT, dan ACL.
  • Membagi Jaringan (Subnetting): Membagi jaringan besar menjadi subnet.
  • Network Address Translation (NAT): Mengubah alamat IP privat menjadi publik.

3. Cara Kerja Router

  1. Menerima Paket Data: Router mendapatkan paket data dari perangkat pengirim.
  2. Membaca Header: Membaca alamat IP tujuan pada header paket.
  3. Cek Tabel Routing: Mencocokkan alamat dengan tabel routing yang dimiliki.
  4. Pemilihan Jalur: Memilih jalur terbaik berdasarkan metrik (hop count, bandwidth).
  5. Pengiriman: Mengirimkan paket ke hop berikutnya menuju tujuan.
  6. Jika Tidak Ditemukan: Paket akan dibuang atau dikembalikan dengan pesan error.

6. Perbandingan dengan Perangkat Jaringan Lain

Perangkat Layer OSI Fungsi Utama Bentuk Addressing
Router Layer 3 (Network) Menghubungkan jaringan berbeda, routing IP Address
Switch Layer 2 (Data Link) Menghubungkan perangkat dalam LAN, forwarding frame MAC Address
Hub Layer 1 (Physical) Meneruskan sinyal ke semua port (broadcast) Tidak ada
Bridge Layer 2 Menghubungkan dua segmen LAN MAC Address

7. Kesimpulan

Router adalah komponen penting dan kritis dalam jaringan modern, berfungsi sebagai:

  • Penghubung jaringan yang berbeda (LAN-WAN, LAN-Internet)
  • Pengatur lalu lintas data melalui proses routing
  • Pengaman jaringan dengan firewall dan NAT
  • Pengelola alamat IP dengan DHCP dan subnetting

2. Keamanan WPA3
3. QoS (Quality of Service)
4. MU-MIMO untuk multi-device
