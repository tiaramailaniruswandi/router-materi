Apa Itu Router?

Router adalah perangkat jaringan cerdas yang beroperasi di Layer 3 model OSI, bertugas sebagai "polisi lalu lintas" data antar jaringan berbeda. Router menentukan jalur optimal untuk paket data mencapai tujuan dengan membaca alamat IP.

Router Enterprise
Cisco, Juniper, MikroTik

Cara Kerja Router

  1. Menerima paket data dari perangkat sumber
  2. Memeriksa header dan alamat IP tujuan
  3. Mencocokkan dengan tabel routing
  4. Memilih jalur terbaik (routing protocol)
  5. Mengirimkan ke hop berikutnya

Fungsi Utama

  • Inter-network Connectivity: Menghubungkan LAN, WAN, Internet
  • Traffic Management: QoS, Load Balancing, Bandwidth Control
  • Security: Firewall, ACL, VPN, NAT
  • Network Segmentation: Subnetting, VLAN Routing
  • Failover & Redundancy: Backup links, dynamic routing
