Router - Fungsi, Cara Kerja, dan Penerapan

1. Pengertian Router

Router adalah perangkat keras jaringan komputer yang berfungsi untuk menghubungkan dua atau lebih jaringan yang berbeda, baik jaringan yang sama media atau berbeda (seperti LAN ke WAN, atau LAN ke internet). Router bekerja pada lapisan Network Layer (Layer 3) dalam model OSI.

Gambar 1: Contoh Router Jaringan

2. Fungsi Router

  • Menghubungkan Jaringan Berbeda: Mengirimkan paket data antar jaringan yang terpisah.
  • Mengatur Lalu Lintas Data (Routing): Menentukan jalur terbaik untuk pengiriman data.
  • Menerapkan Keamanan Jaringan: Dilengkapi fitur firewall, NAT, dan ACL.
  • Membagi Jaringan (Subnetting): Membagi jaringan besar menjadi subnet.

3. Cara Kerja Router

  1. Menerima paket data dari perangkat pengirim
  2. Membaca alamat IP tujuan pada paket data
  3. Mencocokkan alamat dengan tabel routing
  4. Mengirimkan paket ke hop berikutnya
  5. Jika jalur tidak ditemukan, paket akan dibuang

5. Perbandingan dengan Perangkat Lain

Perangkat Layer OSI Fungsi Utama
Router Layer 3 Menghubungkan jaringan berbeda
Switch Layer 2 Menghubungkan perangkat dalam LAN
Hub Layer 1 Meneruskan sinyal ke semua port

6. Kesimpulan

Router adalah komponen penting dan kritis dalam jaringan modern, tidak hanya sebagai penghubung antar jaringan, tetapi juga sebagai pengelola lalu lintas dan pengaman data. Pemahaman tentang router sangat diperlukan bagi administrator jaringan maupun pengguna umum.
